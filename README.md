
# Spot Metrics Challenge #

## Objetivo ##

Permitir a criação de listas de compras segundo os elementos cadastrados.

### Requerimentos ###

* NPM
* Node

### Uso ###

#### Levantando a aplicação

1 - Instale as dependências utilizando NPM
``` sh
  npm install path/to/folder
```

2 - Na pasta do projeto, execute o comando para levantar a aplicação

``` sh
  npm run start
```

#### Enjoy ####
